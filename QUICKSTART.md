# เริ่มใช้ my-ads แบบไม่มีพื้นฐาน (5 นาที)

> อ่านหน้านี้พอ — ไม่ต้องเขียนโค้ดเป็น · แค่ **"พูด" กับ Claude** แล้วกดอนุมัติ
> (อยากรู้ลึก/ติดตั้งละเอียด → `README.md`)

---

## ส่วนที่ 1 — เตรียมของ (ทำครั้งเดียว)

**1. เปิด Terminal**
- Mac: กด `⌘ + Space` → พิมพ์ **Terminal** → Enter
- (Terminal = หน้าจอดำๆ ที่พิมพ์คำสั่ง — ไม่ต้องกลัว)

**2. เอา skill มาไว้ในเครื่อง**
วางคำสั่งนี้ใน Terminal (copy ทั้งบรรทัด → วาง → Enter):
```bash
cp -r ~/Downloads/my-ads ~/.claude/skills/
```
> ปรับ `~/Downloads/my-ads` เป็นที่ที่คุณเก็บโฟลเดอร์ · ถ้าโหลดมาจาก GitHub ก็ใช้ path ที่ clone มา

**3. เปิด Claude Code แล้วพิมพ์**
```
/my-ads
```
เจอ = ติดตั้งสำเร็จ ✅

**4. บอก Claude ให้ช่วยตั้งค่า** (แค่พิมพ์ประโยคนี้)
> **"ช่วยพาตั้งค่า my-ads ทีละขั้น — ทำ token + กรอกข้อมูลธุรกิจ"**

Claude จะพาทำ: สร้าง Token Facebook → ใส่ไฟล์ `.env` → กรอกข้อมูลธุรกิจคุณ (BRAND_PROFILE)
> 🔑 ตอนทำ token บอกว่า **"เอาแบบ 60 วัน"** — ไม่งั้นมันหมดอายุใน 1-2 ชม.

---

## ส่วนที่ 2 — พูดแบบนี้กับ Claude (ไม่ต้องจำโค้ด)

| อยากทำอะไร | พิมพ์แบบนี้ |
|-----------|-----------|
| วางแผนแอด / หากลุ่มลูกค้า | *"ช่วยวางแผนแอด หา persona ให้ธุรกิจฉัน"* |
| หา Interest ยิงกลุ่มไหน | *"หา interest สำหรับกลุ่ม [ชื่อ persona]"* |
| สร้างแคมเปญ | *"สร้างแคมเปญตามแผน (ตั้ง PAUSED ไว้ก่อน)"* |
| ดูผลสัปดาห์นี้ | *"ดูผลแอดสัปดาห์นี้ สรุปให้หน่อย"* |
| แอดแพง อยากปรับ | *"CPR แพงขึ้น ช่วยดูว่าปรับอะไรดี"* |
| สรุปเดือน | *"สรุปผลเดือนนี้ + ลูกค้าถามอะไรบ้าง"* |
| ออกรายงาน PDF | *"ทำรายงานเดือนนี้เป็น PDF"* |
| แอดมีปัญหา/ดึงข้อมูลไม่ได้ | *"แอดใช้ไม่ได้ ช่วยดูให้หน่อย"* |

> Claude จะทำให้ + **ถามยืนยันก่อนทำเรื่องสำคัญ** (เปิดแอด / ใช้เงิน) — คุณแค่กด "โอเค"

---

## ส่วนที่ 3 — จำแค่ 3 กฎ

1. **ก่อนกด "เปิดแอด" ให้ตรวจก่อน** — แอดที่สร้างจะ PAUSED (ยังไม่วิ่ง) เสมอ ปลอดภัย
2. **เรื่องเงิน Claude จะถามยืนยันก่อน** — อย่าเผลอกดผ่านโดยไม่อ่าน
3. **วันไหนแอด "ดึงข้อมูลไม่ได้"** = Token หมดอายุ → พิมพ์ *"ช่วยทำ token ใหม่ 60 วัน"*

---

## ติดตรงไหน?

- ทำ token ไม่เป็น → *"พาทำ Meta token ทีละขั้น ช้าๆ"*
- ไม่รู้ธุรกิจตัวเองใส่อะไร → เปิด `BRAND_PROFILE.md` ดูตัวอย่าง "บ้านในฝัน ดีไซน์"
- อยากเห็นหน้าตา dashboard → เปิด `dashboard-mockup.html` (เป็นตัวอย่าง ยังไม่ใช่ข้อมูลคุณ)
