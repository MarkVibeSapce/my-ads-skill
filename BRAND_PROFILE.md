## 1. ธุรกิจ

| หัวข้อ | ข้อมูล |
|--------|--------|
| ชื่อแบรนด์ | บ้านในฝัน ดีไซน์ (Baan Nai Fun Design) |
| ทำอะไร | รับออกแบบ + สร้างบ้าน ครบวงจร |
| สินค้า/บริการหลัก | ออกแบบบ้าน, รับเหมาก่อสร้าง, บ้านสั่งสร้าง (custom) |
| ราคาเฉลี่ยต่อดีล | 2–5 ล้านบาท/หลัง |
| พื้นที่ให้บริการ | กรุงเทพฯ + นนทบุรี + ปทุมธานี + สมุทรปราการ |
| ช่องทางที่ลูกค้าติดต่อ | ทักแชท Facebook (Messenger) |

### จุดขาย (USP) — ทำไมต้องเลือกเรา
- ออกแบบฟรีถ้าเซ็นสัญญาก่อสร้าง
- มีทีมสถาปนิก + วิศวกรในบริษัท (ไม่จ้างช่วง)
- ผลงานจริง 100+ หลัง มีรีวิว

### ลูกค้ามักติดอะไร (Objection)
- กลัวงบบานปลาย
- กลัวสร้างไม่เสร็จ/ทิ้งงาน
- ไม่รู้จะเริ่มยังไง (มีที่ดินแต่ยังไม่มีแบบ)

## 3. KPI เป้าหมาย

| KPI | ความหมาย | เป้าของเรา |
|-----|----------|-----------|
| **CPR (Cost per Result)** | ค่าต่อ 1 คนทักแชท | **< ฿150** |
| CPM | ค่าต่อ 1,000 การมองเห็น | < ฿120 |
| จำนวนแชท/เดือน | ลูกค้าทักเข้ามา | 60–100 คน |
| นัดคุยจริง/เดือน | แชทที่กลายเป็นนัด | 8–15 นัด |

> **CPR ยิ่งต่ำยิ่งดี** — ถ้าเกิน ฿150 มาก → เปิด `3-optimize.md`

- งบรวม: **฿15,000/เดือน** (~฿500/วัน)
- แบ่ง: ทดสอบช่วงแรก 2 แคมเปญ แคมเปญละ ฿250/วัน

## 4. Persona (จาก Workshop คอร์ส — P3)

> ทำมาจากใบงาน Persona ในคอร์ส · ใช้เป็นกลุ่มเป้าหมาย Ad Set

### Persona 1: "คุณนุ่น" — มีที่ดิน อยากสร้างบ้าน ⭐ (ตัวหลัก)
- อายุ: 35–50 ปี
- อาชีพ/รายได้: เจ้าของกิจการ / พนักงานระดับสูง รายได้ 80k+
- Location: กรุงเทพฯ + ปริมณฑล
- Pain: มีที่ดินแล้ว แต่ไม่รู้จะเริ่มออกแบบยังไง กลัวงบบานปลาย
- Hook: "มีที่ดินแล้ว? ออกแบบบ้านในฝันฟรี เริ่มจากงบที่คุณมี"
- Interest ที่ใช้: อสังหาริมทรัพย์, บ้าน, การลงทุนอสังหา

### Persona 2: "คุณต้น" — คู่แต่งงานใหม่ อยากมีบ้านหลังแรก
- อายุ: 28–38 ปี
- อาชีพ/รายได้: พนักงานบริษัท คู่รัก รายได้รวม 60k+
- Location: กรุงเทพฯ + ปริมณฑล
- Pain: อยากมีบ้านเป็นของตัวเอง กลัวกู้ไม่ผ่าน/ผ่อนไม่ไหว
- Hook: "บ้านหลังแรกของครอบครัว เริ่มต้นวางแผนกับทีมมืออาชีพ"
- Interest ที่ใช้: บ้าน, ครอบครัว, สินเชื่อเพื่อที่อยู่อาศัย

## 6. 🧠 สมองสะสม — สิ่งที่เรียนรู้แล้ว

> ผู้ช่วยเติมส่วนนี้หลังทำงานทุกครั้ง · คนในทีมก็จดเองได้

### สิ่งที่เวิร์ค (ทำต่อ)
- *(ตัวอย่าง)* รูป "บ้านเสร็จจริง + เจ้าของยิ้ม" CPR ต่ำกว่ารูปเรนเดอร์ 3D
- *(ตัวอย่าง)* Persona "คุณนุ่น" CPR ดีสุด — ทุ่มงบกลุ่มนี้ก่อน

### สิ่งที่ไม่เวิร์ค (เลิก/ระวัง)
- *(ตัวอย่าง)* ยิงทั้งประเทศ = CPR แพง → ต้องปักหมุดพื้นที่เสมอ
- *(ตัวอย่าง)* คำว่า "ถูกที่สุด" ในแอด → ได้แต่คนถามราคาแล้วหาย
